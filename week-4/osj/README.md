### 핵심 아이디어

뱀과 사다리를 딕셔너리를 이용해 구현한다.

이미 방문한 번호이면 우선순위에서 밀리기 때문에 제외한다.

<br>

모든 주사위의 경우의 수를 계산하기 위해 bfs를 사용한다. 

<br>

다음 위치를 나타내는 next의 값이 뱀과 사다리 값에 간다면 next 값을 업데이트 해준다.  

<br>

bfs라고 리스트만 사용하는 것이 아니라 캐싱이 필요하다면 딕셔너리를 쓸 수 있도록 노력하면 좋을 것 같다.
